# QA 测试说明文档 — REQ-014 图纸管理用户反馈迭代（APP 端）

> **来源需求**: `requirements/pc/REQ-014-pc.md`
> **产品**: SMART SITE SYSTEM
> **平台**: APP 端（NS Safety App，iOS & Android）
> **测试用例编号格式**: `TC-014-APP-XX`
> **生成日期**: 2026-04-28

---

## 1. 测试目标

覆盖 REQ-014 APP 端 3 条相关反馈（FB-002、FB-005、FB-006），确保：
- SE 在 APP 端可查看和下载附件
- SE 无法在 APP 端看到任何 Markup 入口
- SE 在 APP 端无法看到系统版本号 Ver

---

## 2. 测试范围

| 反馈编号 | 测试模块 | 用例数 |
|----------|----------|--------|
| FB-002 | APP 端版本附件查看 + 下载（SE） | 5 |
| FB-005 | APP 端 SE Markup 入口隐藏 | 4 |
| FB-006 | APP 端 SE Ver 字段隐藏 | 4 |

---

## 3. 测试场景与用例

---

### 3.1 FB-002 APP 端版本附件查看与下载

| 用例 ID | 前置条件 | 操作步骤 | 预期结果 |
|---------|----------|----------|----------|
| TC-014-APP-01 | 以 SE 角色登录 APP；已被分配某图纸版本；该版本在 PC 端已上传至少 1 个附件 | 进入图纸版本详情页，查看附件区域 | 显示"附件（Attachments）"区域，列出所有附件（文件名、类型等）；无上传/删除按钮 |
| TC-014-APP-02 | 同上；附件为 PDF 文件 | 点击附件 [下载] 按钮 | 文件下载后使用系统默认应用打开（iOS 使用 Quick Look / Android 使用文件查看器） |
| TC-014-APP-03 | 同上；附件为 .docx 文件 | 点击附件 [下载] 按钮 | 文件下载成功，弹出可用应用列表供用户选择打开；或提示用户"请在 PC 端查看此类文件"（视实现方案） |
| TC-014-APP-04 | 同上；附件为较大文件（如 50MB） | 点击 [下载] 按钮 | 显示下载进度；下载完成后提示成功 |
| TC-014-APP-05 | 以 SE 角色登录 APP；该版本无附件 | 进入图纸版本详情页，查看附件区域 | 附件区域显示"暂无附件"空态文本 |

---

### 3.2 FB-005 APP 端 SE Markup 入口完全隐藏

| 用例 ID | 前置条件 | 操作步骤 | 预期结果 |
|---------|----------|----------|----------|
| TC-014-APP-06 | 以 SE 角色登录 APP；进入图纸列表页 | 查看图纸列表，检查每行操作入口 | 无 Markup 相关按钮、图标或角标 |
| TC-014-APP-07 | 以 SE 角色登录 APP；进入图纸版本详情页 | 查看详情页所有按钮和入口 | 无 Markup 相关入口（无"查看 Markup"按钮、无 Markup Tab/角标） |
| TC-014-APP-08 | 以非 SE 角色（PM/CM）登录 APP；进入图纸版本详情页 | 查看详情页 | Markup 相关入口正常显示 |
| TC-014-APP-09 | 以 SE 角色登录 APP | 尝试直接访问 Markup 相关接口（通过抓包工具）| 接口返回 403 Forbidden |

---

### 3.3 FB-006 APP 端 SE Ver 字段隐藏

| 用例 ID | 前置条件 | 操作步骤 | 预期结果 |
|---------|----------|----------|----------|
| TC-014-APP-10 | 以 SE 角色登录 APP；进入图纸列表页 | 查看图纸列表卡片内容 | 每张卡片仅显示 Drawing Code，不显示系统版本号（Ver） |
| TC-014-APP-11 | 以 SE 角色登录 APP；进入图纸版本详情页 | 查看详情页字段 | 详情页仅显示 Drawing Code，Ver 字段不存在（无空行占位） |
| TC-014-APP-12 | 以 PM/CM 角色登录 APP；进入同一图纸列表页 | 查看图纸列表卡片内容 | 卡片同时显示 Drawing Code 和 Ver |
| TC-014-APP-13 | 以 PM/CM 角色登录 APP；进入图纸版本详情页 | 查看详情页字段 | 同时显示 Drawing Code 和 Ver 字段 |

---

## 4. 测试执行注意事项

- FB-002 附件下载需在 iOS 和 Android 两个平台分别测试
- FB-002 下载异常场景：网络中断时的处理（重试/提示）需覆盖
- FB-005 需确认 APP 端所有包含 Markup 入口的页面均已完成隐藏
- FB-006 需同时在列表页和详情页验证，确保两处均无 Ver 字段显示
- 测试 SE 视角时需使用已被分配过图纸版本的 SE 账号
