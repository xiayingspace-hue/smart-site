# QA 测试说明文档 — REQ-002 PC 端设备加油记录管理

> **来源需求**: REQ-002-shared + REQ-002-pc
> **平台**: PC 端（Chrome / Edge，1280px+）
> **生成日期**: 2026-04-08

---

## 1. 测试目标

验证 PC 端加油记录管理的增删改查、统计图表、Excel 导出功能的正确性与边界行为。

---

## 2. 测试场景与用例

### 场景 1：新增加油记录

| ID | 前置条件 | 测试步骤 | 预期结果 |
|----|---------|---------|---------|
| TC-002-PC-01 | 已登录，有 `equipment:refueling:create` 权限 | 点击 [New Record]，填写所有必填字段，点击 [Submit] | 记录创建成功，列表刷新，新记录在第一页 |
| TC-002-PC-02 | 同上 | 提交时不填加油量 | 表单校验提示 "Fuel Amount is required" |
| TC-002-PC-03 | 同上 | 上传第 6 张小票照片 | 提示 "Maximum 5 photos allowed" |
| TC-002-PC-04 | 同上 | 填写加油量为负数 | 提示 "Must be a positive number" |

### 场景 2：修改加油记录

| ID | 测试步骤 | 预期结果 |
|----|---------|---------|
| TC-002-PC-05 | 点击任意记录的 [Edit]，修改加油量后保存 | 记录更新，列表中加油量变更 |
| TC-002-PC-06 | 无 `equipment:refueling:update` 权限时 | [Edit] 按钮不显示或禁用 |

### 场景 3：删除加油记录

| ID | 测试步骤 | 预期结果 |
|----|---------|---------|
| TC-002-PC-07 | 点击 [Delete]，弹出确认框后确认 | 记录软删除，列表不再显示，Toast 提示成功 |
| TC-002-PC-08 | 点击 [Delete] 后点取消 | 记录不删除 |
| TC-002-PC-09 | 无 `equipment:refueling:delete` 权限 | [Delete] 按钮不显示 |

### 场景 4：列表筛选与分页

| ID | 测试步骤 | 预期结果 |
|----|---------|---------|
| TC-002-PC-10 | 按设备名关键词搜索 | 仅返回名称含关键词的记录 |
| TC-002-PC-11 | 按燃料类型筛选 `DIESEL` | 仅返回 fuelType=DIESEL 的记录 |
| TC-002-PC-12 | 设置日期范围筛选 | 仅返回 refuelingTime 在范围内的记录 |
| TC-002-PC-13 | 跳转到第 2 页 | 正确显示第 2 页数据 |

### 场景 5：统计页面

| ID | 测试步骤 | 预期结果 |
|----|---------|---------|
| TC-002-PC-14 | 进入统计页，切换分组为 `MONTH` | 图表横轴按月份展示，汇总数据正确 |
| TC-002-PC-15 | 切换分组为 `EQUIPMENT` | 显示各设备加油量排名 |
| TC-002-PC-16 | 设置日期范围 | 图表数据随范围变化 |
| TC-002-PC-17 | 无 `equipment:refueling:statistics` 权限 | 统计菜单不可见或返回 403 |

### 场景 6：导出 Excel

| ID | 测试步骤 | 预期结果 |
|----|---------|---------|
| TC-002-PC-18 | 在列表页点击 [Export]，确认导出 | 下载文件，文件名格式 `Refueling_Records_{projectCode}_{yyyyMMdd}.xlsx` |
| TC-002-PC-19 | 有筛选条件时导出 | 导出仅包含筛选后的所有记录（不受分页限制）|
| TC-002-PC-20 | 无记录时导出 | 下载空表格（含表头）|
| TC-002-PC-21 | 无 `equipment:refueling:export` 权限 | [Export] 按钮不显示 |
