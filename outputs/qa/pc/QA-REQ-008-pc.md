# QA-REQ-008-pc QA测试说明文档

## 1. 需求来源
- 对应需求文档：REQ-008-pc《公司层级工序模板配置》

## 2. 测试范围
- 公司层级工序模板配置页面功能
- 专业Tab、构件类型、工序三级联动
- 工序的增删改查、批量新增、排序
- 工序权重校验
- 工序删除/禁用业务逻辑
- 角色权限控制

## 3. 测试用例

### 3.1 页面入口与布局
| 用例编号 | 测试场景                          | 预期结果                        |
|----------|-----------------------------------|---------------------------------|
| TC-001   | 切换到Company Insights视角        | 左侧显示Project Progress菜单   |
| TC-002   | 点击Project Progress              | 显示Process Settings子菜单     |
| TC-003   | 点击Process Settings              | 显示模板配置页面，含Tab、构件、工序表格 |

### 3.2 专业Tab与构件类型联动
| 用例编号 | 测试场景                          | 预期结果                        |
|----------|-----------------------------------|---------------------------------|
| TC-004   | 切换专业Tab                       | 左侧构件列表和右侧工序表格同步刷新 |
| TC-005   | 切换构件类型                      | 右侧工序表格刷新                |
| TC-006   | 专业Tab/构件类型增删入口          | 无新增/编辑/删除按钮            |

### 3.3 工序增删改查
| 用例编号 | 测试场景                          | 预期结果                        |
|----------|-----------------------------------|---------------------------------|
| TC-007   | 新增工序（正常）                  | 工序添加成功，表格更新          |
| TC-008   | 新增工序（名称重复）              | 提示名称重复，不可提交          |
| TC-009   | 编辑工序                          | 工序信息更新成功                |
| TC-010   | 批量新增工序                      | 多条工序同时添加成功            |
| TC-011   | 工序排序（拖拽）                  | 排序保存成功                    |

### 3.4 工序删除/禁用逻辑
| 用例编号 | 测试场景                               | 预期结果                                    |
|----------|----------------------------------------|---------------------------------------------|
| TC-012   | 删除未被引用的工序                     | 弹出确认，确认后删除成功                    |
| TC-013   | 删除已同步项目模板但未生成实际任务的工序 | 弹出确认（提示同步移除项目级引用），确认后删除成功 |
| TC-014   | 删除已被实际任务使用的工序             | 不显示删除按钮，或提示不可删除              |
| TC-015   | 删除有进度填报历史的工序               | 不显示删除按钮，或提示不可删除              |
| TC-016   | 禁用工序                               | 工序状态变为禁用，行样式灰化，新任务不可引用 |
| TC-017   | 启用已禁用工序                         | 工序恢复启用状态                            |

### 3.5 权重校验
| 用例编号 | 测试场景                          | 预期结果                        |
|----------|-----------------------------------|---------------------------------|
| TC-018   | 权重输入非数字                    | 校验提示，不可提交              |
| TC-019   | 权重总和不等于100%                | 给出提示（非阻断）              |
| TC-020   | 权重输入负数或超过100             | 校验提示，不可提交              |

### 3.6 角色权限
| 用例编号 | 测试场景                          | 预期结果                        |
|----------|-----------------------------------|---------------------------------|
| TC-021   | 企业管理员操作                    | 可新增、编辑、删除、禁用工序    |
| TC-022   | Planning角色操作                  | 仅可查看，操作按钮不可见或置灰  |
| TC-023   | PM角色操作                        | 仅可查看，操作按钮不可见或置灰  |

## 4. 边界与异常测试
- 工序名称超长字符输入
- 大量工序数据下的列表性能
- 并发操作同一工序（删除/编辑）
- 网络异常时操作反馈

## 5. 验收标准
- 所有正常流程用例通过
- 权限控制符合需求
- 删除/禁用业务逻辑与需求一致
- 界面无明显UI缺陷
