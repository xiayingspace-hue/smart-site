# UI说明文档补充 — Company Insights视角下Project Progress菜单

## 1. 入口与交互

- 用户点击PC端右上角项目选择器（🟠 项目名 ▼），在下拉菜单中选择“进入公司视角（Company Insights）”。
- 切换后，页面主区域进入公司层级视图，顶部/侧边栏高亮显示当前为“公司层级”。

## 2. 左侧导航栏

- 在Company Insights视角下，左侧导航栏显示“Project Progress”菜单项（建议图标：📊）。
- 当前点击“Project Progress”菜单后，主内容区显示空白页面（后续根据业务推进补充具体内容）。

## 3. 设计要点

- “Project Progress”菜单项风格与其他一级菜单一致，支持选中高亮。
- 切换回项目视角后，导航栏恢复为项目相关菜单。
- 保持整体UI一致性，切换动画与现有菜单切换保持一致。

## 4. 交互流程示意

1. 用户点击右上角项目选择器
2. 选择“进入公司视角（Company Insights）”
3. 左侧导航栏出现“Project Progress”菜单
4. 点击后主内容区为空白
5. 用户可随时切回项目视角

---

> 备注：本说明仅涉及PC端UI，移动端暂不涉及公司层级切换。
