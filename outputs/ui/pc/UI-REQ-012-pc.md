# UI-REQ-012-pc.md

## CM"我的任务"PC 端 UI 说明

> 对应需求：REQ-012
> 平台：PC 端（Vue2 + Element UI）
> 影响页面：CM 我的任务列表页、工序任务创建侧滑弹框、任务详情页（Tab 结构）、问题汇报弹框

---

### 1. 设计目标

- 为 CM 提供专属的任务列表与任务执行页面，风格与 PM 端保持一致
- 操作列直接提供"新建工序任务"与"汇报问题"入口，减少层级跳转
- 任务详情页以 Tab 结构区分信息类型（只读信息 / 可操作信息），结构清晰
- 工序任务创建侧滑弹框支持引用模板和自定义两种模式，权重字段突出显示

---

### 2. 页面清单

| 页面 / 组件 | 类型 | 说明 |
|-------------|------|------|
| CM 我的任务列表页 | 全页面 | 筛选区 + 任务表格 |
| 工序任务创建侧滑弹框 | 侧滑 Drawer | 从操作列"新建工序任务"触发 |
| 工序任务编辑侧滑弹框 | 侧滑 Drawer | 从任务详情"工序任务" Tab 编辑按钮触发，与创建弹框复用 |
| 任务详情页 | 全页面 | Tab 结构：任务信息 / 工序任务 / 问题上报 |
| 问题汇报弹框 | Dialog | 从操作列"汇报问题"触发 |

---

### 3. 页面详细说明

---

#### 3.1 CM 我的任务列表页

**整体布局（与 PM 端一致）：**

```
┌──────────────────────────────────────────────────────────────┐
│  Progress Management > 我的任务（CM）                        │
├──────────────────────────────────────────────────────────────┤
│  [状态筛选 ▼]  [WBS 筛选 ▼]  [时间范围 ▼]  [搜索框]  [重置] │
├──────────────────────────────────────────────────────────────┤
│  Activity ID │ Activity Name │ WBS │ 计划时间 │ Deviation │   │
│  Status │ 优先级 │ 操作                                      │
├──────────────────────────────────────────────────────────────┤
│  ACT-001     │ xxxxxx        │ ... │ ...      │ ...       │   │
│                              │   [新建工序任务] [汇报问题]  │
│  ACT-002     │ xxxxxx        │ ... │ ...      │ ...       │   │
│                              │   [新建工序任务] [汇报问题]  │
├──────────────────────────────────────────────────────────────┤
│  共 xx 条   < 1 2 3 ... >                                    │
└──────────────────────────────────────────────────────────────┘
```

**表格字段：**

| 字段 | 说明 |
|------|------|
| Activity ID | 任务唯一编号 |
| Activity Name | 可点击，进入任务详情页 |
| WBS | WBS 编号 |
| 计划开始/结束时间 | 格式 YYYY-MM-DD |
| 实际开始/结束时间 | 有数据时显示 |
| Deviation | 偏差天数，超期显示红色 |
| Status | 任务状态标签，颜色与 PM 端一致 |
| 优先级 | High / Medium / Low，彩色标签 |
| 进度 | 百分比，系统自动计算（只读） |
| 操作 | 新建工序任务、汇报问题 |

**操作列说明：**
- **新建工序任务**：主要操作，蓝色文字按钮；点击后从页面右侧弹出侧滑弹框
- **汇报问题**：次要操作，灰色文字按钮；点击后弹出问题汇报 Dialog
- 操作列按权限控制显示/置灰（当前版本 CM 均有权限）

#### 3.2 工序任务创建 / 编辑侧滑弹框

**触发方式：**
- 创建：任务列表操作列"新建工序任务"
- 编辑：任务详情"工序任务" Tab 内点击某条工序任务的"编辑"按钮

**弹框布局：**

```
┌──────────────────────────────────────────────────────────────┐
│  新建工序任务（/ 编辑工序任务）                    [×]       │
├──────────────────────────────────────────────────────────────┤
│  关联上级任务：ACT-001 xxxxxxxx（只读）                       │
├──────────────────────────────────────────────────────────────┤
│  创建方式：[● 引用工序模板]  [○ 自定义新建]                  │
├──────────────────────────────────────────────────────────────┤
│  ── 引用模板时显示 ──                                         │
│  [按专业筛选 ▼] [按工序类型筛选 ▼] [按区域筛选 ▼]            │
│  ┌─────────────────────────────────────────────────────────┐ │
│  │ ☑ 模板名称A   | 权重：5  | 专业：结构  | 类型：浇筑    │ │
│  │ ☑ 模板名称B   | 权重：3  | 专业：建筑  | 类型：砌筑    │ │
│  │ ☐ 模板名称C   | 权重：4  | 专业：机电  | 类型：管道    │ │
│  └─────────────────────────────────────────────────────────┘ │
│  [确认引用选中模板]                                           │
├──────────────────────────────────────────────────────────────┤
│  ── 工序任务列表（批量编辑区域）──                            │
│  工序名称* │ WBS │ 计划开始* │ 计划结束* │ 优先级* │ 权重* │  │
│  分配SE*  │ 备注  │ 操作                                     │
│  ─────────────────────────────────────────────────────────  │
│  [工序名A]  [WBS] [日期]     [日期]     [High ▼] [5    ]    │
│  [SE-001 ▼]      [备注...]  [删除]                           │
│  ─────────────────────────────────────────────────────────  │
│  [+ 添加一行]                                                 │
├──────────────────────────────────────────────────────────────┤
│                                    [取消]  [保存]            │
└──────────────────────────────────────────────────────────────┘
```

**字段说明：**

| 字段 | 必填 | 说明 |
|------|------|------|
| 关联上级任务 | — | 只读，自动带出当前行任务 |
| 工序任务名称 | ✅ | 手动输入 |
| WBS | 可选 | 手动输入 |
| 计划开始时间 | ✅ | 日期选择器 |
| 计划结束时间 | ✅ | 日期选择器，不早于开始时间 |
| 优先级 | ✅ | High / Medium / Low 下拉 |
| 权重 | ✅ | 数字输入，范围 1～10；引用模板时自动带出可修改，自定义时必填 |
| 分配 SE | ✅ | 下拉选择 SE 用户 |
| 备注 | 可选 | 文本域 |

**权重字段交互：**
- 引用模板：自动填入模板预设权重，输入框可编辑，边框提示"来自模板，可修改"
- 自定义新建：空白输入框，带占位符"请输入 1～10"，超出范围时显示红色提示
- 权重不填或为 0 时，保存按钮不可用

**页面布局：**

```

│  （对应 Tab 内容）                                           │
└──────────────────────────────────────────────────────────────┘
```

---

##### Tab 1：任务信息（只读）

展示当前任务的所有基本字段，所有字段均只读，使用 Element UI Descriptions 组件布局：

| 字段 | 显示 |
|------|------|
| Activity ID | 文本 |
| Activity Name | 文本 |
| WBS | 文本 |
| 计划开始/结束时间 | 文本 |
| 实际开始/结束时间 | 文本（无数据显示"—"） |
| Deviation | 文本，超期红色 |
| Status | 状态标签 |
| 优先级 | 彩色标签 |
| 进度 | 百分比 + 进度条（系统计算，只读） |
| 备注 | 文本 |

---

##### Tab 2：工序任务（可编辑）

```
┌──────────────────────────────────────────────────────────────┐
│  工序任务列表                                                │
├──────────────────────────────────────────────────────────────┤
│  工序名称 │ 权重 │ 计划时间 │ 优先级 │ 分配 SE │ 状态 │ 操作 │
├──────────────────────────────────────────────────────────────┤
│  土方开挖  │  3   │ ...      │ High   │ SE-001  │已完成│ [编辑]（置灰） │
│  基础浇筑  │  5   │ ...      │ High   │ SE-002  │已完成│ [编辑]（置灰） │
│  墙体砌筑  │  2   │ ...      │ Medium │ SE-001  │未完成│ [编辑]         │
├──────────────────────────────────────────────────────────────┤
│  任务进度：80%（8/10）                                        │
└──────────────────────────────────────────────────────────────┘
```

**交互说明：**
- 已完成（锁定）的工序任务行整体置灰，"编辑"按钮 disabled，Tooltip 提示"该工序已标记完成并锁定"
- 未完成的工序任务"编辑"按钮可点击，点击后从右侧弹出编辑侧滑弹框
- 底部显示当前任务整体进度（公式：已完成权重之和 / 全部权重之和 × 100%），系统自动计算，只读

---

##### Tab 3：问题上报（只读）

展示 SE 在该任务下上报的所有问题记录：

| 字段 | 说明 |
|------|------|
| 问题类型 | 分类标签 |
| 问题描述 | 文本 |
| 上报人 | SE 姓名 |
| 上报时间 | 日期时间 |
| 处理状态 | 待处理 / 处理中 / 已解决 |
| 附件 | 下载链接 |

- 所有字段只读，CM 不可在此页面操作

---

#### 3.4 问题汇报弹框（Dialog）

**触发方式：** 任务列表操作列"汇报问题"

```
┌──────────────────────────────────────┐
│  汇报问题                    [×]     │
├──────────────────────────────────────┤
│  关联任务：ACT-001（只读）           │
│  问题类型*：[下拉选择 ▼]             │
│  问题描述*：[文本域]                  │
│  影响范围：[文本域]                   │
│  建议方案：[文本域]                   │
│  附件：[上传文件]（可选）             │
├──────────────────────────────────────┤
│                    [取消]  [提交]     │
└──────────────────────────────────────┘
```

- 提交后自动发送通知给对应 PM
- 提交成功后 Dialog 关闭，列表无需刷新

---

### 4. 设计约束

- 整体风格与 PM 端"我的任务"页面一致（Element UI 组件、色彩规范、字体、间距）
- 侧滑弹框宽度建议 600～800px，支持内容区域滚动
- 操作列按钮采用文字按钮（`el-button type="text"`），多个按钮用分隔符 `|` 隔开
- 已完成锁定行背景色使用浅灰色（`#F5F5F5`），文字颜色降低对比度（`#AAAAAA`）
- 进度条使用 Element UI `el-progress`，百分比显示在进度条右侧
