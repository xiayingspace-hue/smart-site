# UI-REQ-012-pc.md

## CM"我的任务"PC 端 UI 说明

> 对应需求：REQ-012
> 平台：PC 端（Vue2 + Element UI）
> 影响页面：CM 我的任务列表页、工序任务创建侧滑弹框、任务详情页（Tab 结构）、问题汇报弹框

---

### 1. 设计目标

- 为 CM 提供专属的任务列表与任务执行页面，风格与 PM 端保持一致
- 操作列直接提供"新建工序任务"与"汇报问题"入口，减少层级跳转
- 任务详情页以 Tab 结构区分信息类型（只读信息 / 可操作信息）
- 工序任务 Tab 展示三态状态（Not Started / In Progress / Completed），Completed 行置灰
- 工序任务创建侧滑弹框支持引用模板和自定义两种模式，权重字段突出显示

---

### 2. 页面清单

| 页面 / 组件 | 类型 | 说明 |
|-------------|------|------|
| CM 我的任务列表页 | 全页面 | 筛选区 + 任务表格 |
| 工序任务创建侧滑弹框 | 侧滑 Drawer | 从操作列"新建工序任务"触发 |
| 工序任务编辑侧滑弹框 | 侧滑 Drawer | 从任务详情"工序任务" Tab 编辑按钮触发，与创建弹框复用 |
| 任务详情页 | 全页面 | Tab 结构：任务信息 / 工序任务 / 问题上报 |
| 问题汇报弹框 | Dialog | 从操作列"汇报问题"触发 |

---

### 3. 页面详细说明

---

#### 3.1 CM 我的任务列表页

**整体布局（与 PM 端一致）：**

```
┌──────────────────────────────────────────────────────────────┐
│  Progress Management > 我的任务（CM）                        │
├──────────────────────────────────────────────────────────────┤
│  [状态筛选 ▼]  [时间范围 ▼]  [搜索框]  [重置]              │
├──────────────────────────────────────────────────────────────┤
│  Activity ID │ Activity Name │ 计划时间 │ Deviation │          │
│  Status │ 优先级 │ 进度 │ 操作                              │
├──────────────────────────────────────────────────────────────┤
│  ACT-001  │ xxxxxx  │ ...  │ ...  │ ...  │ In Progress │ ... │
│                        │ [详情] [新建工序任务] [汇报问题]  │
│  ACT-002  │ xxxxxx  │ ...  │ ...  │ ...  │ Completed   │ ... │
│                   │ [详情] [新建工序任务(灰)] [汇报问题]   │
├──────────────────────────────────────────────────────────────┤
│  共 xx 条   < 1 2 3 ... >                                    │
└──────────────────────────────────────────────────────────────┘
```

**操作列按钮规则：**

| 按钮 | 可点击条件 | 置灰条件 |
|------|-----------|---------|
| 详情 | 始终可点击 | — |
| 新建工序任务 | 任务状态为待开始 / 进行中 / 逾期 | 任务状态为已完成或已作废 |
| 汇报问题 | 任务状态非已作废 | 任务状态为已作废 |

**详情弹框说明：**

- 点击"详情"按钮，从右侧弹出任务详情侧滑弹框（Drawer），宽度建议 800px
- 弹框内容与 §3.3 任务详情页一致，采用相同 Tab 结构（任务信息 / 工序任务 / 问题上报）
- 弹框不跳转页面，关闭后回到列表页，列表数据自动刷新

> 任务已有工序任务时，仍可继续追加新工序（无数量上限）

#### 3.2 工序任务创建 / 编辑侧滑弹框

**弹框布局：**

```
┌──────────────────────────────────────────────────────────────┐
│  新建工序任务（/ 编辑工序任务）                    [×]       │
├──────────────────────────────────────────────────────────────┤
│  关联上级任务：ACT-001 xxxxxxxx（只读）                       │
├──────────────────────────────────────────────────────────────┤
│  创建方式：[● 引用工序模板]  [○ 自定义新建]                  │
├──────────────────────────────────────────────────────────────┤
│  ── 引用模板时显示 ──                                         │
│  [按专业筛选 ▼] [按工序类型筛选 ▼] [按区域筛选 ▼]            │
│  ┌─────────────────────────────────────────────────────────┐ │
│  │ ☑ 模板名称A  | 权重：5  | 专业：结构  | 类型：浇筑     │ │
│  │ ☑ 模板名称B  | 权重：3  | 专业：建筑  | 类型：砌筑     │ │
│  └─────────────────────────────────────────────────────────┘ │
│  [确认引用选中模板]                                           │
├──────────────────────────────────────────────────────────────┤
│  ── 工序任务列表（批量编辑区域）──                            │
│  工序名称* │ 计划开始* │ 计划结束* │ 优先级* │ 权重*       │ │
│  分配SE*  │ 备注 │ 操作                                      │
│  [工序名A] [日期]   [日期]    [High ▼]  [5   ]              │
│  [SE-001 ▼]     [备注...]  [删除]                            │
│  [+ 添加一行]                                                 │
├──────────────────────────────────────────────────────────────┤
│                                    [取消]  [保存]            │
└──────────────────────────────────────────────────────────────┘
```

**字段说明：**

| 字段 | 必填 | 说明 |
|------|------|------|
| 关联上级任务 | — | 只读，自动带出 |
| 工序任务名称 | ✅ | 手动输入 |
| 计划开始时间 | ✅ | 日期选择器 |
| 计划结束时间 | ✅ | 不早于开始时间 |
| 优先级 | ✅ | High / Medium / Low |
| 权重 | ✅ | 范围 1～10；引用模板自动带出可改，自定义必填 |
| 分配 SE | ✅ | 下拉选择 |
| 备注 | 可选 | 文本域 |

---

##### Tab 1：任务信息（只读）

所有字段只读，使用 Element UI Descriptions 组件布局：

| 字段 | 说明 |
|------|------|
| Activity ID / Name | 文本 |
| 计划开始/结束时间 | 文本 |
| 实际开始/结束时间 | 文本（无数据显示"—"） |
| Deviation | 文本，超期红色 |
| Status | 状态标签 |
| 优先级 | 彩色标签 |
| 进度 | 百分比 + 进度条（系统计算，只读） |
| 备注 | 文本 |

---

##### Tab 2：工序任务（可编辑）

```
┌──────────────────────────────────────────────────────────────────────┐
│  工序名称 │ 权重 │ 计划时间 │ 实际开始 │ 实际结束 │ 优先级 │ SE │ 状态 │ 操作 │
├──────────────────────────────────────────────────────────────────────┤
│  土方开挖  │  3   │ ...     │ 06-01   │ 06-05   │ High  │SE-1│Completed │ [编辑](灰) │
│  基础浇筑  │  5   │ ...     │ 06-05   │ —       │ High  │SE-2│In Progress│ [编辑](灰) │
│  墙体砌筑  │  2   │ ...     │ —       │ —       │ Medium│SE-1│Not Started│ [编辑]     │
├──────────────────────────────────────────────────────────────────────┤
│  任务进度：80%（8/10）                                                │
└──────────────────────────────────────────────────────────────────────┘
```

**状态标签样式：**

| 状态 | 标签颜色 |
|------|---------|
| Not Started | 灰色 `#909399` |
| In Progress | 蓝色 `#409EFF` |
| Completed | 绿色 `#67C23A` |

**行交互规则：**
- **Completed 行**：整体背景 `#F5F5F5`，文字 `#AAAAAA`，编辑按钮 disabled，Tooltip 提示"该工序已完成并锁定"
- **In Progress / Not Started 行**：编辑按钮可点击，点击后从右侧弹出编辑侧滑弹框
- 底部实时展示任务进度（Completed 权重之和 / 全部权重之和 × 100%），只读

---

##### Tab 3：问题上报（只读）

| 字段 | 说明 |
|------|------|
| 问题类型 | 分类标签 |
| 问题描述 | 文本 |
| 上报人 | SE 姓名 |
| 上报时间 | 日期时间 |
| 处理状态 | 待处理 / 处理中 / 已解决 |

---

### 4. 设计约束

- 整体风格与 PM 端"我的任务"页面一致（Element UI 组件、色彩规范）
- 侧滑弹框宽度建议 600～800px，内容区域可滚动
- 操作列按钮采用 `el-button type="text"`，多个按钮以 `|` 分隔
- Completed 行背景 `#F5F5F5`，文字 `#AAAAAA`
- 进度条使用 `el-progress`，百分比显示在进度条右侧

---

### 5. 验收条件

- [ ] 操作列"新建工序任务"按钮在任务已完成/已作废时置灰
- [ ] 侧滑弹框支持引用模板和自定义，权重字段校验 1～10
- [ ] 工序任务 Tab 展示三态状态标签（Not Started / In Progress / Completed）
- [ ] 工序任务 Tab 展示实际开始/结束时间列（系统自动记录，只读）
- [ ] Completed 行整体置灰，编辑按钮 disabled
- [ ] Tab 底部进度公式计算正确并实时更新
- [ ] 问题汇报 Dialog 提交后关闭
