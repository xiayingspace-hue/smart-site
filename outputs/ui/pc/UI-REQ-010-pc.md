# UI说明文档 — REQ-010 项目进度管理菜单与任务分配页面布局

## 1. 页面入口与菜单结构
- 视角：项目管理后台
- 左侧导航栏：Progress Management
- Progress Management 下有两个菜单：
  - 甘特图：点击后进入项目甘特图页面（当前显示空白，无需开发）
  - Master Program：点击后进入Planning任务分配页面

## 2. Master Program页面结构

### 2.1 顶部操作区
- 包含“Filter Search”筛选按钮、批量操作按钮、右上角“新增”按钮
- 支持多选、批量分配、批量作废等操作

### 2.2 任务分配表格
- 字段包括：
  - Activity ID
  - Activity Name
  - WBS
  - Plan Start/End（如 S: 23-03-2026 E: 23-04-2024）
  - Actual Start/End（如 S: 23-03-2026 E: 23-04-2024）
  - Deviation（如“+522 +546”，正负号区分提前/延后，红色=延期，绿色=提前，灰色=如期）
  - Duration
  - Created By
  - Assigned PM
  - Status（未分配、已分配、进行中、已完成、已延期、作废，颜色高亮）
  - Operation（分配、编辑、作废等按钮）
- 支持分页、筛选、排序、搜索
- 作废任务行样式为淡灰色/删除线，操作按钮禁用，状态高亮“作废”

### 2.3 操作按钮交互
- **分配按钮**：
  - 未分配或可重新分配任务显示，点击弹出分配弹框（选择PM、备注、确认/取消）
  - 分配成功后表格更新，Toast提示
  - 不可分配时按钮置灰/隐藏，悬停提示原因
- **编辑按钮**：
  - 可编辑任务显示，点击弹出编辑弹框（可编辑字段：名称、优先级、计划时间、负责人、备注等）
  - 编辑成功后表格更新，Toast提示
  - 已完成、作废、锁定等任务按钮置灰/隐藏，悬停提示原因
- **作废按钮**：
  - 未完成、未作废、未被下级分解/锁定任务显示，点击弹窗二次确认并填写作废原因
  - 作废后状态高亮，行样式变化，所有操作禁用

### 2.4 新增/编辑弹框
- 右侧侧滑弹出，布局参考现有系统风格
- 字段包括：Activity Name、Priority、Plan Start/End、Assigned To、备注等
- 底部为“Cancel”和“Save”按钮

### 2.5 状态与偏差展示
- Status字段用不同颜色/标签高亮
- Deviation字段数字+颜色，正负号区分提前/延后，红色=延期，绿色=提前，灰色=如期

### 2.6 详情页与作废说明
- 点击任务可进入详情页，作废任务顶部高亮“已作废”，显示作废原因、作废人、作废时间
- 只读信息可查看，所有操作禁用

### 2.7 其他
- 支持筛选/统计作废任务
- 所有操作均有权限校验和Toast提示
- 保持与现有管理后台风格一致（Element UI风格，主色#4A90D9）
- 建议仅支持PC端大屏，1280px及以上
