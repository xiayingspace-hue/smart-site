# UI-REQ-014-app.md

## 图纸管理用户反馈迭代 UI 说明（APP 端）

> 对应需求：REQ-014（REQ-003/004/005 第一版用户反馈汇总）
> 平台：APP 端（NS Safety App）
> 影响范围：FB-002（SE 附件下载）、FB-005（SE 角色隐藏 Markup 相关入口）

---

### 1. 设计目标

| 反馈编号 | 涉及需求 | APP 端改动摘要 |
|----------|----------|----------------|
| FB-002 | REQ-003 | SE 在 APP 端查看图纸版本详情时，可下载附件（Attachments） |
| FB-005 | REQ-004 | SE 角色 APP 端完全隐藏 Markup 相关入口 |
| FB-006 | REQ-003 / REQ-005 | SE 在 APP 端查看图纸时，只显示 Drawing Code，隐藏系统版本号（Ver） |

> FB-001 / FB-003 / FB-004 为 PC 端改动，APP 端无对应改动。

---

### 2. 改动点说明

---

#### 2.1 FB-002 SE 在 APP 端可下载图纸版本附件

**涉及位置：**
- APP 端图纸版本详情页（已分配 SE 可见的版本详情）

**交互说明：**
- 版本详情页中，PDF 主文件下方展示附件（Attachments）列表区域
- 附件列表展示：文件名、类型、大小、备注
- SE 可点击附件条目进行下载，下载权限与 PDF 下载权限一致（已分配版本即可下载）
- SE **不显示**上传附件按钮和删除按钮
- 若该版本暂无附件，附件区域显示空状态提示：`"暂无附件"`

**APP 端版本详情页结构示意（SE 视角）：**

```
┌──────────────────────────────────┐
│  图纸版本详情            [关闭]  │
├──────────────────────────────────┤
│  Drawing Code：DWG-001           │
│  Ver：1   上传时间：2026-04-01   │
│                                  │
│  [查看 PDF]  [下载 PDF]          │
├──────────────────────────────────┤
│  附件（Attachments）             │
│  ┌────────────┬──────┬─────────┐ │
│  │ 文件名     │ 类型 │  操作   │ │
│  ├────────────┼──────┼─────────┤ │
│  │ xxx.dwg    │ CAD  │  下载   │ │
│  │ xxx.docx   │ 文档 │  下载   │ │
│  └────────────┴──────┴─────────┘ │
└──────────────────────────────────┘
```

---

#### 2.2 FB-005 SE 角色完全隐藏 Markup 相关入口（APP 端）

**涉及位置：**
- APP 端图纸列表页（如有 Markup 入口）
- APP 端图纸版本详情页（如有 Markup 相关区域）

**权限规则：**

| UI 元素 | 非SE角色（PM / CM） | SE角色 |
|---------|---------------------|--------|
| 图纸列表"Markup 历史"入口 | ✅ 显示 | ❌ 隐藏 |
| 图纸详情中 Markup 相关区域 | ✅ 显示 | ❌ 隐藏 |
| Markup 数量角标 / 红点 | ✅ 显示 | ❌ 隐藏 |

**说明：**
- SE 登录后查看图纸列表及详情时，不出现任何 Markup 相关的按钮、角标或区域
- 隐藏后页面布局保持正常，不留空白占位
- APP 端隐藏逻辑与 PC 端一致，统一由角色权限控制

---

#### 2.3 FB-006 SE 视角隐藏系统版本号（Ver），仅显示 Drawing Code（APP 端）

**涉及位置：**
- APP 端图纸列表页（版本字段）
- APP 端图纸版本详情页（版本信息区）

**各角色字段展示差异：**

| 字段 | PM / CM | SE |
|------|---------|-----|
| Drawing Code | ✅ 显示 | ✅ 显示 |
| 系统版本号（Ver） | ✅ 显示 | ❌ 隐藏 |

**SE 端版本详情页示意（结合 2.1 附件区域）：**

```
┌──────────────────────────────────┐
│  图纸版本详情            [关闭]  │
├──────────────────────────────────┤
│  Drawing Code：DWG-001           │
│  上传时间：2026-04-01            │
│                                  │
│  [查看 PDF]  [下载 PDF]          │
├──────────────────────────────────┤
│  附件（Attachments）             │
│  ...                             │
└──────────────────────────────────┘
```

**说明：**
- SE 视角中系统 Ver 字段完全不渲染，不留空白占位，布局保持紧凑

---

### 3. 各角色 APP 端视角差异总结

| 功能 | PM / CM | SE |
|------|---------|-----|
| 附件列表查看 | ✅ | ✅（已分配版本） |
| 附件下载 | ✅ | ✅（已分配版本） |
| 附件上传 / 删除 | ✅（有权限者） | ❌ |
| Markup 历史入口 | ✅ | ❌ 隐藏 |
| Markup 相关区域 | ✅ | ❌ 隐藏 |
| 系统版本号（Ver）显示 | ✅ | ❌ 隐藏 |

---

### 4. 验收条件

- [ ] SE 角色 APP 端图纸版本详情中展示附件列表，可下载，无上传/删除按钮；暂无附件时显示空状态
- [ ] SE 角色 APP 端图纸列表和详情中不出现任何 Markup 相关入口、按钮或角标，隐藏后布局无空白占位
- [ ] SE 角色 APP 端图纸列表及版本详情中不显示系统 Ver 字段，仅展示 Drawing Code；隐藏后布局无空白占位

---

> 本 UI 说明与 REQ-014 需求文档配套，APP 端涉及 FB-002 附件下载、FB-005 Markup 权限调整、FB-006 系统版本号隐藏，具体 UI 细节以设计稿为准。
