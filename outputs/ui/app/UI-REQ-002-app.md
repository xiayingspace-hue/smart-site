# UI 说明文档 — APP 移动端设备加油记录

> **来源需求**: [REQ-002-app](../../../requirements/app/REQ-002-app.md) + [REQ-002-shared](../../../requirements/shared/REQ-002-shared.md)
> **产品**: SMART SITE SYSTEM
> **平台**: APP 移动端（iOS & Android，UNIAPP + Vue 2）
> **设计令牌参考**: [UI-REQ-001-app § 2](./UI-REQ-001-app.md)
> **生成日期**: 2026-03-23
> **最后更新**: 2026-04-02

---

## 1. 设计目标

- 为工地设备管理员提供一套简洁高效的加油信息录入界面，减少现场录入负担
- 列表页能快速定位和浏览加油记录，支持按设备、日期、燃油类型筛选
- 新增/编辑表单设计要兼顾信息完整性与录入效率，减少不必要的输入步骤
- 详情页清晰展示所有加油信息，支持凭证照片预览
- 视觉风格与首页保持一致，延续 REQ-001-app 设计体系
- 支持多语言（中文 / English），默认英文界面

## 5. 屏幕适配要求

| 维度 | 规格 |
|------|------|
| 最小支持屏幕 | 4.5 英寸（约 320px 宽） |
| 最大支持屏幕 | 6.7 英寸（约 428px 宽） |
| 设计稿基准宽度 | 375px |
| 适配方式 | rpx（UNIAPP 响应式单位）或百分比布局 |
| 方向 | 仅竖屏 |
| iOS Safe Area | 顶部状态栏 + 底部 Home Indicator 区域需适配 |
| Android 状态栏 | 适配不同机型状态栏高度 |
| 长表单适配 | 新增/编辑页使用 ScrollView，键盘弹出时自动上推当前聚焦字段 |
| 小屏适配 | 设备图片缩小至 60×60px；照片缩略图缩小至 72×72px |
| 大屏适配 | 卡片和表单最大宽度不超过 428px，居中显示 |
