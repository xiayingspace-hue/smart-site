# UI 说明文档 — APP 移动端登录和首页

> **来源需求**: [REQ-001-app](../../../requirements/app/REQ-001-app.md) + [REQ-001-shared](../../../requirements/shared/REQ-001-shared.md)
> **产品**: SMART SITE SYSTEM
> **平台**: APP 移动端（iOS & Android，UNIAPP + Vue 2）
> **生成日期**: 2026-03-23
> **最后更新**: 2026-04-02

---

## 1. 设计目标

- 为工地管理人员提供一个简洁、专业、可信赖的移动端登录入口
- 首页（Work Space）以项目为核心，快速触达各业务功能模块
- 视觉风格现代、清爽，契合智慧工地产品定位
- 支持多语言（中文 / English），默认英文界面
- 确保在 iOS 和 Android 双平台上保持一致的视觉和交互体验

---

## 2. 页面清单与用户流程

### 2.1 页面清单

| 序号 | 页面 | 说明 |
|------|------|------|
| 1 | 登录页 | 品牌展示 + 登录表单 |
| 2 | 首页（Work Space） | Banner + 功能模块卡片 |
| 3 | 项目选择面板 | 左侧滑出的项目列表 |
