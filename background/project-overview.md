# 项目概述

## 项目名称

Smart Site System (智慧工地管理系统)

## 项目简介

Smart Site System 是一个专为建筑工地打造的数字化集成管理平台。系统通过物联网（IoT）和大数据技术，将人员管理、机械设备管理、物料管理、项目进度管理、环境管理、安全管理、质量管理、车辆进出场管理、塔吊使用申请管理、工地文档管理、工地反馈管理及 BIM 模型渲染等模块深度整合，旨在实现施工现场的全方位数字化监控与智能决策支持。

整个平台分为三个端，不同端的设计规范不一样
H5移动网页端：用户不需要登录，直接访问相关的链接可以快速申请不同类型的业务表单，例如车辆进出场申请、车辆载物出场申请等
NS Safety App端：用户需要输入账号和密码登录app，然后可以访问权限下的系统功能
管理PC端：用户需要输入账号和密码登录管理端，然后可以访问权限下的系统功能

## 项目目标

### 长期目标
首个项目标杆: 在公司内部的RWS建筑项目完成 Smart Site System 的全面落地与验证。
规模化推广: 将成熟的方案推广至公司旗下的第二个及更多建筑项目。
行业输出: 将系统产品化，向外部建筑公司及合作伙伴推广，输出数字化工地管理能力。

### 短期目标（当前版本）
人员管理: 实现实名制考勤、人员定位及工种配比分析。
机械设备管理: 监控塔吊、升降机等重型机械的运行状态与维护周期。
安全管理: 建立安全巡检闭环系统，利用 AI 识别违规行为。
质量管理: 数字化质量验收流程，实现问题照片留痕与整改追踪。
车辆进出管理: 自动识别进出场车辆，记录物料运输频次。

## 项目愿景

通过构建“施工现场数字孪生”，消除建筑行业传统的信息滞后与管理盲区，打造安全、绿色、高效的未来工地，成为行业领先的智慧建筑解决方案供应商。

## 项目范围

### 包含内容
- 多端访问: 整个系统涉及到三个端，移动网页端（H5端）、移动端App现场作业、PC管理端，三个端的设计规范不一样
- 政府对接: 自动对接新加坡相关政府监管系统（如实名制数据上报）
- 数据可视化: 提供实时驾驶舱大屏，展示项目进度、安全及环境监测指标

### 不包含内容（Out of Scope）
- 财务与工资核算: 仅提供考勤原始数据，不直接处理复杂的税务或工资计算。
- 采购供应链: 专注于现场物料使用与进场，不涉及外部供应商的招投标流程。
