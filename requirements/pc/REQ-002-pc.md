> **端**: PC 管理端（桌面浏览器 Web 管理后台）
> **共享需求**: [REQ-002-shared.md](../shared/REQ-002-shared.md)（业务规则、数据模型、API 接口）

## 基本信息

- **需求ID**: REQ-002-pc
- **需求标题**: PC 端设备加油数据管理与统计
- **产品**: SMART SITE SYSTEM
- **平台**: PC 管理端（桌面浏览器）
- **技术框架**: Vue 2 + Element UI
- **优先级**: 中
- **状态**: 草稿

## 需求描述

### 背景与目标

PC 管理端作为数据管理和决策支持端，主要面向项目经理和设备管理员。在设备加油管理功能中，PC 端承担数据查看、高级筛选、统计分析和数据导出的职责。APP 端录入的加油数据实时同步至 PC 端展示。

### 用户故事

```
作为 项目经理
我想要 在 PC 管理后台查看所有设备的加油记录和油耗统计报表
以便 我能掌握项目的燃油成本，分析各设备的油耗情况，及时发现异常消耗
```

```
作为 设备管理员（PC 端）
我想要 能够筛选、查看、编辑和导出加油记录，必要时可补录或删除错误数据
以便 我能确保加油数据的准确性和完整性
```

- 鼠标悬浮行高亮

#### 2.3 筛选区域

- 所有筛选条件默认为空（显示全部数据）
- Equipment 为下拉选择器，数据来源于当前项目的设备列表
- Subcontractor 为下拉选择器，数据来源于当前项目的分包商列表
- Fuel Type、Method 为枚举下拉
- Date Range 为日期范围选择器
- Creator 为下拉选择器，数据来源于当前项目的用户列表
- Supplier 为文本输入，模糊搜索
- Cost Range 为两个数字输入框（最低 ~ 最高）
- [Search] 按钮触发查询
- [Reset] 按钮清除所有筛选条件并刷新

#### 2.4 统计摘要卡片

列表页顶部显示 4 张统计摘要卡片，数据根据当前筛选条件动态计算：
- **Total Fuel (L)**: 筛选结果的加油量总和
- **Total Cost**: 筛选结果的费用总和
- **Record Count**: 筛选结果的记录条数
- **Avg Cost / Record**: 平均每条记录费用

> 注意：摘要数据需后端提供独立接口，或在分页查询接口中附带统计字段，避免前端全量加载计算。

#### 2.5 批量操作

- 表头勾选框全选当前页
- 选中多条记录后，底部显示浮动操作栏：
  - "Delete Selected (N)" — 批量删除（需删除权限）
- 批量删除前显示确认弹窗

#### 5.3 统计功能说明

**筛选条件**:
- Date Range：默认最近 3 个月
- Equipment：可选某一台设备或全部
- Subcontractor：可选某一分包商或全部
- Fuel Type：可选某种燃油类型或全部
- Group By：分组维度，默认 Month

**汇总指标卡片**:
- 根据筛选条件动态计算
- 数字使用千分位格式显示

**趋势图表**:
- 柱状图 + 折线图混合（柱状图为加油量，折线图为费用）
- X 轴根据 Group By 切换（日 / 周 / 月 / 设备 / 分包商）
- 鼠标悬浮显示详细数值 Tooltip
- 支持图表类型切换（柱状图 / 折线图）

**设备油耗排行**:
- 水平条形图，按加油量降序
- 显示百分比占比
- 默认显示 Top 10，其余归入 "Others"

**分包商油耗排行**:
- 水平条形图，按加油量降序
- 显示百分比占比和关联设备数量
- 默认显示 Top 10，其余归入 "Others"
- 点击分包商名可展开查看其下属设备明细

**明细数据表格**:
- 与图表数据对应
- 支持排序

### 6. 数据导出

- 列表页右上角 [📥 Export] 按钮
- 导出当前筛选条件下的所有记录（不限分页）
- 导出格式为 Excel（.xlsx）
- 导出字段：设备编号、设备名称、设备类型、所属分包商、加油日期、加油时间、燃油类型、加油量(L)、单价、总费用、表读数、加油方式、供应商、备注、创建人、创建时间
- 导出文件名格式：`Refueling_Records_{项目名}_{导出日期}.xlsx`
- 大数据量导出（> 5000 条）使用异步导出 + 下载通知
